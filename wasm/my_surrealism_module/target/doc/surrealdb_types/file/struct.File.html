    pub key: String,

Expand description

Represents a file reference in SurrealDB

A file reference points to a file stored in a bucket with a specific key. This is used for file storage and retrieval operations.

Fields§

§bucket: String

The bucket name where the file is stored

§key: String

The key/identifier for the file within the bucket
