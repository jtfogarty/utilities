pub fn count<R: RangeBounds<String>>(range: R) -> Result<u64>
Expand description

Counts the number of key-value pairs within a specified key range.

This function transfers the range bounds to the runtime via FFI and returns the count of all key-value pairs whose keys fall within the specified range.

§Type Parameters

  • R: A type that implements RangeBounds<String> for defining the key range.

§Parameters

  • range: The range of keys to count (e.g., "a".."z" or .."prefix").

§Returns

A Result containing a u64 representing the count of key-value pairs within the range, or an error if the operation fails.

§Errors

  • If transferring data fails.
  • If the FFI call or result reception encounters an issue.
